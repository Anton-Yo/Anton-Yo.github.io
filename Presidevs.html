Protect the Presidevs

In yet another game jam, my team and I sought to push outside of our comfort zone, creating Protect the Presidevs. We decided to create the game in Godot with zero experience, as it was garnering attention due to Unity's pricing changes. Despite this, we were rated #1 overall at the 2023 MACS Online Game Jam. Play on itch.io.

My Contributions:

  • Animation
    • Using sprites sourced from the internet, I used Godot's built in animation tool to create a "Monkey Throw" and "Monkey Idle" animation
    • Used GDscript to rotate between the animations as needed
  • Monkey
    • Developed a system for the monkey to randomly throw an object at 6 different enemies on cooldown
  • Developing UI
    • Assisted with developing the start and end screen UI
    • Implemented the code to switch between the UI and reset the game state on restart
