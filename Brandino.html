The Brandino

The Brandino is a 10 to 15 minute long browser based murder-mystery set on a pirate ship. Inspired by Lucas Pope's 'Obra Dinn', the player must gather information throughout the ship to accuse one of three crewmates as the murderer. Play the game here.

Features:

  • Three coding languages
    • Used HTML & CSS to design the layout of each scene
    • Used Javascript (jQuery) for the gameplay of each scene
  • Various Forms of Gameplay
    • Implemented a quiz, button-spam game, a 2 round dice game that randomly creates html dice elements, and interviews with dialogue
  • Notebook
    • Used two Javascript files to create a notebook, one containing the HTML code, and one for Javascript functionality
    • Automatically changes the notebook information according to the user's progress. I used local storage to carry progress between scenes
    • Contains a customisable note section, where user can input and manage notes, preserving their progress through local storage
  • Photoshop
    • Edited all the pictures in Photoshop for use in the project
