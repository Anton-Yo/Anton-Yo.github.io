Basement Game

Basement game is a 2.5D automation game in which you must create elements by 'magic', or in reality, using scientific machines to create a production line. It was developed over the course of two weeks as part of Pirate Software's Game Jam. I acted as an on-call developer role during this game jam, employing my expertise in a couple different areas and learning how to better use Unity's animator. Play on itch.io.

My Contributions:

  • Animation
    • Implemented animation for 7 different machines
    • Used Unity's animator to manipulate gameobjects and particle systems to create natural looking animations
    • Implemented a lightbulb on each machine, using lighting/emissive materials to create visual feedback for player
  • Audio
    • Sourced and implemented all the audio components, with pitch variation
  • Programming
    • Implemented the objective system, which reads objectives from a JSON file, and recognises when the objective has been met
    • Added simple camera movement
