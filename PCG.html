Procedural Dungeon Generator

As part of a major project at university, I decided to research and tackle procedural generation. My goal was to make a highly customisable and developer friendly tool. To achieve this, I adapted a binary partitioning algorithm in Unity, which uses 16 different parameters. The generator outputs a dungeon layout according to these parameters. A video going through the project is included on the right.

Breakdown:

  • Binary Partitioning Algorithm
    • Researched numerous strategies for procedurally generating dungeon layouts, such as Cellular Automata
    • Decided upon a binary partioning algorithm, where a certain space is divided in half. The children rectangles are then divided in half repeatedly
  • Features
    • Divides an area of customisable width and height into divisions using a using recursion
    • Randomises rooms in those divisions according to minimum and maximum values
    • Generates pathways between rooms, based on the parent-child structure of the generated rooms.
    • Can specify a number of or percent chance for double size rooms
    • Includes a toggleable debug UI visualise each aspect of generation
  • Playtesting
    • Created a Google Forms survey and conducted a playtest with 5 developers
    • Incoporated user feedback, like 'Number of Rooms' text into next iteration
