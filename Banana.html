Banana Facts Website

For an assignment at university, I made a simple website about bananas. As my first proper attempt of web development, there are a few issues. However, I'm still proud of it as a result of my research and creativity when designing the website layout. The website can be viewed here.

Features:

  • Three coding languages
    • Used HTML and CSS for page layout
    • Used jQuery to manage what elements are displayed
  • 5 different page layouts
    • Created a banana history timeline with navigation buttons to bring up different time periods
    • Used CSS animations to make 4 boxes containing fun banana-related facts to rotate in a circle
    • Elements that change when hovered over
    • Learnt and utilised CSS flexbox
